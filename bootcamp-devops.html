Bootcamp DevOps 3 Bulan

Program intensif selama 3 bulan untuk menguasai DevOps dan infrastruktur modern. Dari dasar Linux hingga deployment Kubernetes, dengan proyek hands-on dan mentoring langsung.

Kurikulum Bootcamp

Linux Fundamentals

Penguasaan dasar Linux, shell scripting, dan manajemen sistem.

Git & Version Control

Teknik branching, merging, dan kolaborasi dengan Git.

CI/CD Pipelines

Implementasi continuous integration dan deployment dengan tools seperti Jenkins atau GitHub Actions.

Docker Containerization

Membuat, mengelola, dan mengoptimalkan container Docker.

Kubernetes Orchestration

Deployment, scaling, dan manajemen aplikasi di Kubernetes cluster.

Monitoring & Logging

Setup monitoring dengan Prometheus, Grafana, dan ELK Stack.

Ansible Automation

Automate konfigurasi dan deployment dengan Ansible.

Terraform IaC

Infrastructure as Code untuk provisioning cloud resources.

GitOps with FluxCD & ArgoCD

Implementasi GitOps untuk continuous delivery menggunakan FluxCD dan ArgoCD pada Kubernetes.

Google Cloud Platform

Deployment infrastruktur DevOps di GCP, termasuk GKE, Cloud Build, dan IAM.

Amazon Web Services

Manajemen cloud dengan AWS: EKS, CodePipeline, ECS, dan services terkait DevOps.
